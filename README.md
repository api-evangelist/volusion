# Volusion

Volusion is an all-in-one e-commerce platform that has powered over 180,000 online businesses since 1999. The platform provides merchants with tools for building and managing online stores, including website hosting, shopping cart software, payment processing, inventory management, and API integrations.

## API

The Volusion API enables merchants on Business and Prime plans to connect third-party applications with their Volusion stores. The API supports import and export of core e-commerce data including products, categories, customers, orders, shipping, and inventory.

- **API Version**: 2.0
- **Authentication**: Admin credentials (email + encrypted password) or OAuth (Volt platform)
- **Base URL**: `https://yourstore.com/api/2.0`
- **Access**: Business plan ($299/month) or Prime (custom pricing) required
- **Documentation**: https://help.volusion.com/s/thevolusionapi

### Supported Resources

- Products (create, update, delete, retrieve)
- Orders (retrieve, update status)
- Customers (create, update, retrieve)
- Categories
- Inventory
- Shipping
